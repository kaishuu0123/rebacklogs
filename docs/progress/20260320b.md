### ⑥ ブランドカラー設定機能（Admin テーマ設定）

tweakcn の 42 テーマプリセットを application_settings で設定できる機能を実装。

#### アーキテクチャ

```
設定保存:  admin が ThemePicker で選択 → フォーム送信 → Setting.theme_name に保存
CSS 注入:  layouts/application.html.erb の <head> で theme_style_tag を呼び出し
          → ThemeHelper が config/theme_presets.json を読んで :root { --primary: ...; } を出力
```

#### 新規ファイル

- `config/theme_presets.json` — tweakcn 42 テーマの light モード CSS 変数（hex 形式）
  - フォント変数・シャドウ変数は除外（色変数のみ）
  - Node.js スクリプトで `/tmp/theme-presets.ts` から自動生成
- `app/javascript/lib/theme-previews.ts` — React ピッカー用の軽量プリセットデータ
  - key, label, primary, background, secondary, muted, accent の 7 プロパティのみ
- `app/javascript/components/react/ThemePicker.tsx` — テーマ選択 UI コンポーネント
  - 42 テーマ × カラーチップ 4 つ（primary / accent / secondary / muted）+ ラベル
  - 選択中のテーマには右上にチェックマーク
  - 「Default (shadcn)」ボタンでテーマなし状態に戻せる
  - `<input type="hidden" name="settings[theme_name]" value="...">` を DOM に出力し、親 ERB フォームと連携
- `app/helpers/theme_helper.rb` — CSS 変数生成ヘルパー
  - `ThemeHelper.css_variables_for(theme_name)` → `--primary: #xxx; ...` の文字列
  - `theme_style_tag` インスタンスメソッド（ビュー/レイアウトで使用）

#### 変更ファイル

- `app/models/setting.rb` — `field :theme_name, type: :string, default: ''` を追加
- `app/controllers/application_settings/dashboard_controller.rb` — `settings_params` に `:theme_name` を追加
- `app/views/layouts/application.html.erb` — `<%= theme_style_tag %>` を `<head>` 内に追加
- `app/views/application_settings/dashboard/index.html.erb` — テーマセクションを追加、`turbo_mount "ThemePicker"` で React コンポーネントを埋め込み
- `app/javascript/entrypoints/application.ts` — `ThemePicker` を turbo-mount に登録
- `app/javascript/locales/en.json` / `ja.json` — `settings.theme.default` を追加

#### 仕様

- 対象: admin ロールのみ（`authorize_resource class: false` で保護済み）
- ダークモード: 未対応（light のみ）
- フォント変数: 未適用（フォント読み込みが必要なため除外）
- テーマ数: 42（tweakcn 準拠）
- DEMO モード: 未実装（ユーザー指示で延期）
