### ③ Biome セットアップ（linter + formatter）

**採用理由**: VitePlus（https://viteplus.dev/）と比較検討した結果、
VitePlus は `vite_rails` gem との統合実績が不明なため Biome を採用。

#### 追加
- `@biomejs/biome 2.4.7` (devDependencies)

#### 設定ファイル
`biome.json`:
- 対象: `app/javascript/entrypoints/**` と `app/javascript/components/react/**`
- インデント: スペース 2
- クォート: シングル
- trailing commas: on
- import 自動ソート: on

#### npm scripts
```json
"lint":     "biome check ...",
"lint:fix": "biome check --write ...",
"format":   "biome format --write ..."
```

#### 検証
```
yarn lint     →  Checked 3 files. No fixes applied.（エラーなし）
yarn lint:fix →  Fixed 3 files.（セミコロン・import 順など自動修正）
```
